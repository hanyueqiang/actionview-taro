## How to install

环境要求：
> 安装 node ~v12.8.1, npm ~6.10.2

下载代码：  
> git clone https://github.com/lxerxa/actionview-taro.git  
> cd actionview-taro  
> npm install  

运行Demo：  
> npm run dev:h5  
> http://localhost:10086/
